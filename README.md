# 🍽️ Menu App

A stylish cross-platform Flutter application for browsing restaurant or café menus. Featuring smooth animations, star ratings, elegant custom fonts, and a clean UI — built to make food look as good as it tastes.

---

## ✨ Features

- 📋 **Menu browsing** — Browse food and drink items in a clean, organized layout
- ⭐ **Star ratings** — Interactive rating system powered by `flutter_rating_bar`
- 🎨 **Custom typography** — Beautiful Rigatoni & Playfair Display fonts for a premium feel
- 🌀 **Loading animations** — Smooth spinners via `flutter_spinkit`
- 🌐 **Network support** — Fetches data via HTTP for dynamic content
- 📱 **Cross-platform** — Runs on both Android and iOS

---

## 🛠️ Tech Stack

| Tool | Purpose |
|---|---|
| Flutter & Dart | Cross-platform UI framework |
| `google_fonts` | Extended font options |
| `flutter_rating_bar` | Star rating widget |
| `flutter_spinkit` | Loading animations |
| `http` | Network/API requests |

## 🚀 Getting Started

### Prerequisites
- [Flutter SDK](https://flutter.dev/docs/get-started/install) (^3.9.0)
- Android Studio or VS Code with Flutter extension

### Installation

1. Clone the repository:
```bash
   git clone https://github.com/NW3N4R/menu_app.git
   cd menu_app
```

2. Install dependencies:
```bash
   flutter pub get
```

3. Run the app:
```bash
   flutter run
```
